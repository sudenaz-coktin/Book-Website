# Codex Anima

Codex Anima is a small web project created to practice frontend development while combining books with visual and interactive elements.

## About the Project
The project focuses on presenting books through a simple and immersive user interface. 
It includes a gallery-style layout, category filtering, hover interactions, and an "About Us" page that explains the idea behind the platform.

The goal was to improve my skills in layout design, UI effects, and basic JavaScript interactions.

## Pages
- **Home / Gallery:** Displays books with hover and modal-style interactions  
- **My Library:** Allows organizing selected books  
- **About Us:** Explains the concept, mission, and vision of Codex Anima  

## Features
- Responsive navigation bar with dropdown menu
- Category filtering structure
- Hover effects and UI animations
- Simple JavaScript for user interaction
- About page with structured content and design-focused layout

## Technologies Used
- HTML5
- CSS3
- JavaScript
